# SKK MIGAS TEST
This project is a test for SKK MIGAS Recruitment done using Java

## Prerequisites
- Java 17 or higher
- Maven 3 or higher

## Getting Started
1. Clone this repository.
   ```
   git clone git@github.com:vigorjs/rekomendasos.git
   ```

2. Navigate to the project folder.
    ```
   cd Skk-Migas-Test
   ```

3. Navigate to the Class Folder
    ```
   cd src/main/java/
   ```

4. Build the project.
    ```
   javac Main.java
   ```

5. Run the project.
    ```
   java Main
   ```
